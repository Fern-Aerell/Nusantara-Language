# Nusantara File Extension (.nusa)

Pertamakali saya mau menentukan ekstensi yang cocok untuk bahasa pemrograman ini, saya bingung, tapi waktu itu saya mempunyai beberapa kadidat yang mungkin akan menjadi file extension dari bahasa ini.

| Nama Ekstensi   | Asalnya            |
| --------------- | ------------------ |
| ~~.NST~~        | NuSanTara          |
| ~~.NS~~         | NuSantara          |
| ~~.NT~~         | NusanTara          |
| ~~.NL~~										| Nusantara Language |
| ~~.NUSANTARA~~			| Nusantara											|
|	~~.NSA~~									|	NuSantarA											|
|	~~.NSL~~									|	NuSantara Language		|
| .NUSA												|	NUSAntara											|

Saya sudah mengecek semua list ekstensi di atas dan yang unik dan tidak dikenali adalah ```nusantara``` dan ```nusa```, namu karena ```nusantara``` terlalu panjang, akhir nya saya memutuskan untuk menggunakan ```nusa``` sebagai ekstensi file dari bahasa pemrograman ini.
