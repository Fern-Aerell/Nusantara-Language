# Nusantara File Extension

Saya sebenarnya masih bingung untuk nama file extension dari bahasa pemrograman ini, tapi saya mempunyai beberapa kadidat yang mungkin akan menjadi file extension dari bahasa ini.

| Nama Ekstensi   | Asalnya            |
| --------------- | ------------------ |
| .NST            | NuSanTara          |
| .NS             | NuSantara          |
| .NT             | NusanTara          |
| .NL (saat ini)  | Nusantara Language |

Jika kalian ada ide yang lebih bagus bisa berkontribusi ke github ini.
