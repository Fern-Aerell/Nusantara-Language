# Operasi Nusantara

Berikut adalah operasi yang bisa dilakukan beserta arah dan urutan eksekusi nya.

### Operasi Notasi Unary (dari kanan ke kiri)

- ```++``` (Pre Increment)  
Contoh : ++x

- ```--``` (Pre Decrement)  
Contoh : --x

- ```~``` (Bitwise NOT)  
Contoh : ~x

- ```!``` (Logical NOT)  
Contoh : !x

### Operasi Peningkatan/Penurunan (dari kanan ke kiri)

- ```++``` (Post Increment)  
Contoh : x++

- ```--``` (Post Decrement)  
Contoh : x--

### Operasi Matematika (dari kiri ke kanan)

Perkalian dan Pembagian  
- ```*``` (Multiplication)  
Contoh : x * y

- ```/``` (Division)  
Contoh : x / y

- ```%``` (Modulus)  
Contoh : x % y

Penjumlahan dan Pengurangan
- ```+``` (Addition)  
Contoh : x + y

- ```-``` (Subtraction)  
Contoh : x - y
