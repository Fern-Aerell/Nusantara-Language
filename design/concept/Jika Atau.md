# Jika Atau

Nusantara memiliki jika atau, ini sama
seperti if else di bahasa pemrograman lain.

## Jika (If)

```nusantara
bulat nilai = 10;
jika(nilai > 75) {
    cetak("Lulus!");
}
```

## Atau Jika (Else If) 

```nusantara
bulat nilai = 10;
jika(nilai > 75) {
    cetak("Lulus!");
}atau jika(nilai == 0) {
    cetak("Tinggal kelas");
}
```

## Atau (Else) 

```nusantara
bulat nilai = 10;
jika(nilai > 75) {
    cetak("Lulus!");
}atau jika(nilai == 0) {
    cetak("Tinggal kelas");
}atau {
    cetak("Tidak Lulus!");
}
```
