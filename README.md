## Nusantara dibuat oleh
- Nusantara di desain oleh Fern Aerell.
- Dan dikembangkan oleh Fern Aerell.

## Sejarah
- Nusantara pertamakali di kembang kan pada tahun 2023.

## Kenapa bahasa ini dibuat
- Nusantara di buat sebagai bagaian dari pembelajaran.
- Mencoba membuat bahasa pemrograman dengan bahasa Indonesia.

## Rencana
- Sintaks nya mudah untuk di pelajari, apalagi untuk orang indonesia.
- Multi platform.
- Bisa digunakan untuk membuat
	- Aplikasi console.
	- Aplikasi web.
	- Aplikasi mobile.
	- Aplikasi desktop.
	- Aplikasi server.
- Bisa berkomunikasi dengan bahasa lain, atau bisa di integrasikan dengan bahasa lain.
- Bisa di konvert ke bahasa lain.
- NullSafety.
- Performa Cepat.
- Mendukung package dan plugin.
